# Twitter integration

## Data tracked

Activities

* Follows
* Tweets with mentions
* Tweets with custom-picked hashtag

Historical import

* Up to 10,000 tweets with mentions/custom-picked hashtags from the past

Not tracked

* Unfollows

Refresh period

* 30 minutes
